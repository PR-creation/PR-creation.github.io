Armstrong number in c


PROGRAM: 



#include
int main()
{
	int a,b,c,sum=0;
	printf("Enter a number:");
	scanf("%d",&a);
	b=a;
	while(b!=0){
		c=b%10;
		sum+=c*c*c;
		b/=10;
	}
	if(sum==a)
	 printf("It is an Armstrong number");
	else
	 printf("It is not an Armstrong number");
}

OUTPUT:



Enter a number:153
